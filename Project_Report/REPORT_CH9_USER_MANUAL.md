# 📖 Chapter 9: Operational Manual — User Guide

This chapter details the operational manual for the VEBStore platform, providing a guide for both customers and administrators.

## 9.1 Customer Boutique Guide (Frontend)
- **Account Creation**: Quick signup with email verification.
- **Shopping Pipeline**:
  - Browse high-contrast product grid.
  - Select sizes and add to 'Smart Bag'.
  - Unified checkout: Choose 'COD' or 'Secure Razorpay'.
- **Post-Purchase Support**:
  - Live tracking via 'Order' page.
  - One-click order cancellation for non-shipped items.
  - Professional return/exchange request system (within 7 days).
- **Communication Alerts**: Receive automated email confirmations (Nodemailer) and WhatsApp alerts (Simulator).

## 9.2 Administrative Management Guide (Management Hub)
- **Inventory Control**:
  - Add products with 4-slot image upload (Cloudinary).
  - Edit existing items via 'Quick Editor' in lists.
  - Instant visual feedback on current stock.
- **Operational Dashboard**:
  - Track total revenue from delivered orders.
  - Real-time order processing through the 'Status Selector'.
- **Response Command Center**:
  - Approve/Reject customer return and exchange requests.
  - Respond to customer reviews.
- **System Global Settings**:
  - Exclusive theme switching (Classic Light / Signature Dark).
  - Administration Shield (2FA Status) and Backup dispatcher controls.
