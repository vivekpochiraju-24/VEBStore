## 🌐 Netlify Deployment (404 Fix)
If you are seeing a 404 error on Netlify when refreshing the page, ensure you have correctly configured the **Base Directory** in your Netlify site settings:

1. **For Frontend**: 
   - Base directory: `frontend`
   - Build command: `npm run build`
   - Publish directory: `dist`
2. **For Admin**: 
   - Base directory: `admin`
   - Build command: `npm run build`
   - Publish directory: `dist`

*The `_redirects` and `netlify.toml` files are already included in the `public` folders to handle SPA routing.*
